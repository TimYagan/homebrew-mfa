# homebrew-mfa

Official Homebrew tap for [mfa](https://github.com/TimYagan/mac-file-analyzer), a fast macOS file analyzer written in Rust.

## Installation

```bash
brew tap TimYagan/mfa
brew install mfa
```

## Upgrade

```bash
brew upgrade mfa
```

## Uninstall

```bash
brew uninstall mfa
brew untap TimYagan/mfa
```

## Usage

```
mfa [OPTIONS] [PATH]
```

See `mfa --help` for the full list of flags, or visit the [mac-file-analyzer](https://github.com/TimYagan/mac-file-analyzer) repo for detailed documentation.
